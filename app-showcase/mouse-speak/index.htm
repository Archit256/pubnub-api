Start Typing to Talk with other Mice
Don't put this script on a page where User Input is requred.
  1. Channel: 
    mouse-speak
  2. Message: 
    { uuid : uuid, pos: [x,y], txt: "hello" }
  3. Events: 
    mousemove, touchmove, touchstart, keyup
  4. Wait State to prevent network bashing.
  5. Send what has changed (mouse, text).
